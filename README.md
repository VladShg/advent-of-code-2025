# Advent of Code 2025

Solutions for Advent of Code 2025 challenges.

## Setup


```bash
npm install
```

## Running Solutions

```bash
npx tsx solutions/day<number>.ts
```

### Examples

```bash
npm run solution

npx tsx solutions/day1.ts

npx tsx solutions/day4.ts
```
